# Flappy Bird Clone - Made using Claude Code

A simple Flappy Bird clone built with Python and Pygame. This is a dummy project used to play around with Claude Code and get more familiar with it as a tool.

## Installation

1. Install the required dependencies:
```bash
python -m venv venv
venv/bin/python -m pip install -r requirements.txt
```

## How to Play

1. Run the game:
```bash
venv/bin/python flappy_bird.py
```

2. Controls:
   - Press **SPACE** to make the bird jump
   - Press **SPACE** to restart after game over
   - Press **ESC** to quit

## Game Features

- Smooth bird physics with gravity
- Randomly generated pipe obstacles
- Score tracking
- Collision detection
- Game over and restart functionality

## Gameplay

- Navigate the bird through the gaps between pipes
- Each pipe you pass increases your score
- Don't hit the pipes, ground, or ceiling!

Enjoy playing!
